# Billing Document Type

Represents the type of billing document this is: either an `invoice`, a `credit memo` or a `debit memo`.

## Enumeration

`BillingDocumentType`

## Fields

| Name | Description |
|  --- | --- |
| `INVOICE` | An invoice. |
| `CREDIT_MEMO` | A credit memo. |
| `DEBIT_MEMO` | A debit memo. |
